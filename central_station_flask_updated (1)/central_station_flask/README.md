# Центральный вокзал — Flask-приложение

## Запуск в PyCharm
1. Откройте папку `central_station_flask` как проект.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Запустите файл `app.py`.
4. Перейдите по ссылке из консоли: `http://127.0.0.1:5000/`.

Данные пользователей, билетов и маршрутов сохраняются в файл `data/railway_data.dat` через `pickle`, как в консольной лабораторной работе.
